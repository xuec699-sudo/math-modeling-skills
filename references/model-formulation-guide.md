# 数学模型表述规范指南

> **适用**: 所有数模论文的模型建立章节。Agent 在 `model_N_build` 和 `content_assembly`
> 阶段**必须**遵循本指南。未达标的模型表述将被 `quality_gate.py model_formulation` 拦截。

---

## 一、通用原则

### 1.1 模型表述必须回答的四个问题

| 问题 | 要求 | 反例 (FAIL) |
|------|------|------------|
| **这是什么模型？** | 明确模型类型：优化/统计/微分方程/分类/预测/评价/图论/... | "建立了数学模型" — 没说类型 |
| **变量是什么？** | 每个变量列出：符号、中文名、单位、取值范围 | Y = -0.4953 + 0.00126W — W 没说是什么 |
| **公式怎么来的？** | 每个公式给出推导过程或文献依据 | 直接丢出公式无任何解释 |
| **假设是什么？** | 列出模型成立的前提条件 | 未提及任何假设 |

### 1.2 符号表规范

每个模型必须包含符号说明表，格式如下：

```
表X  模型符号说明
| 符号 | 含义 | 单位 | 类型 |
|------|------|------|------|
| W    | 孕妇孕周数 | 周 | 自变量 |
| B    | 孕妇BMI | kg/m² | 自变量 |
| Y    | 胎儿Y染色体浓度 | % | 因变量 |
| β₀   | 截距项 | - | 待估参数 |
```

### 1.3 公式编号规范

- 每个公式独立编号：(1), (2), (3)...
- 文中引用格式："如式(3)所示"
- 多行公式用 (3a), (3b) 或对齐环境

## 二、按模型类型的表述模板

### 2.1 优化模型

**必须包含的要素**:

```
1. 决策变量定义
   - 列出所有决策变量，说明物理含义和单位
   - 说明变量类型（连续/整数/0-1）

2. 目标函数
   - 写出完整数学表达式
   - 说明优化方向（min/max）
   - 解释目标函数的物理/经济含义

3. 约束条件
   - 每个约束写出数学表达式
   - 解释每个约束的现实含义
   - 区分硬约束和软约束

4. 模型类型声明
   - 如：线性规划(LP)、整数规划(IP)、混合整数规划(MILP)、
         非线性规划(NLP)、多目标优化等
```

**标准模板**:

```
X.X 模型建立

[问题背景描述，说明为什么这是一个优化问题]

决策变量：
  x_i: [变量的物理含义]，i = 1,2,...,n，[变量类型，如连续/0-1]
  [如有多个变量，逐一列出]

目标函数：
  min Z = Σᵢ c_i x_i + Σⱼ d_j y_j           (1)
  其中：c_i 表示[含义]，d_j 表示[含义]
  [解释目标函数每一项的业务含义]

约束条件：
  ① [约束名称]: Σᵢ a_{ki} x_i ≥ b_k,  k = 1,...,K     (2)
     含义：[该约束的业务解释]
  ② [约束名称]: x_i ∈ {0,1},  i = 1,...,n              (3)
     含义：[变量类型的解释]
  ③ [非负约束]: x_i ≥ 0,  i = 1,...,n                  (4)

综上，建立如下[模型类型]：
  min Z = Σᵢ c_i x_i
  s.t.  [列出所有约束条件]
        x_i ≥ 0

模型假设：
  (1) [假设1及合理性说明]
  (2) [假设2及合理性说明]
```

### 2.2 统计/回归模型

**必须包含的要素**:

```
1. 因变量和自变量定义（含单位）
2. 模型函数形式（线性/非线性/对数/多项式）
3. 参数含义解释
4. 误差项分布假设
5. 模型选择的理由
```

**标准模板**:

```
X.X 模型建立

[说明研究目标和变量关系]

变量定义：
  因变量 Y: [含义]，单位：[单位]
  自变量 X₁: [含义]，单位：[单位]
  自变量 X₂: [含义]，单位：[单位]
  ...

模型形式选择：
  考虑到[理由，如散点图显示的线性趋势/理论依据/参考文献]，
  建立如下[线性/非线性]回归模型：

  Y = β₀ + β₁ X₁ + β₂ X₂ + ... + βₚ Xₚ + ε      (1)

  其中：
  β₀ 为截距项，表示[解释其含义]
  βᵢ (i=1,...,p) 为回归系数，表示 Xᵢ 每增加1单位时 Y 的平均变化量
  ε 为随机误差项，假设 ε ~ N(0, σ²)

模型假设：
  (1) 线性性：Y 与 Xᵢ 之间为线性关系
  (2) 独立性：各观测值相互独立
  (3) 同方差性：误差项方差恒定
  (4) 正态性：误差项服从正态分布
  [检验结果和违反时的处理说明]
```

4. 初始条件/边界条件
5. 模型机理说明（为什么用这个方程）

状态变量：

  I(t): t时刻的[含义]，单位：[单位]
  [列出所有状态变量]

模型机理：
  基于[物理定律/传播机理/文献依据]，系统的动态变化遵循以下规律：
  - [机理1的描述及数学表达]
  - [机理2的描述及数学表达]
  - ...

控制方程：
  dS/dt = [表达式]                              (1)
    含义：[方程右端每一项的物理解释]

    含义：[方程右端每一项的物理解释]
  ...

参数说明：
  α: [参数名]，取值 α = [值]，来源：[文献/数据估计/经验值]
  β: [参数名]，取值 β = [值]，来源：[文献/数据估计/经验值]
  ...

初始条件：
  S(0) = S₀, I(0) = I₀, ...                    (n)

边界条件（如适用）：
  [写出边界条件及物理含义]

### 2.7 物理模型（干涉/衍射/波动/热传导/扩散）

**适用场景**:
- 光学干涉与衍射问题（薄膜干涉、Fabry-Perot干涉、双光束干涉、光栅衍射）
- 波动传播问题（声波、电磁波、弹性波）
- 热传导与扩散问题（Fourier定律、Fick定律）
- 电磁场问题（Maxwell方程组、静电场/静磁场）
- 力学系统（Newton力学、Lagrange/Hamilton力学）
- 流体力学问题（Navier-Stokes方程、Bernoulli方程）

**必须包含的要素**:

```
1. 物理量定义
   - 列出所有物理量：符号、中文名、单位、物理意义
   - 区分已知量（参数）和未知量（待求解变量）
   - 标注矢量/标量/张量类型

2. 控制方程/物理定律
   - 写出完整的控制方程（干涉条件/波动方程/扩散方程/...）
   - 标注每一项的物理含义和单位
   - 说明方程的理论来源（Fresnel公式/Snell定律/Airy公式/...）
   - 对于干涉模型：光程差(OPD)表达式 + 干涉条件 + 折射角公式

3. 参数定义与取值依据
   - 每个参数：符号、物理含义、单位、取值、取值依据
   - 区分：已知参数（查文献/手册）、待测参数（模型求解目标）、假设参数

4. 边界条件/初始条件
   - 写出完整的初始条件和边界条件
   - 说明条件选取的物理依据

5. 模型假设与适用范围
   - 列出所有假设，说明其物理合理性和局限性
   - 判断是否符合实际的物理场景
```

**模板A：光学干涉模型（薄膜/多层膜/Fabry-Perot）**

```
X.X 模型建立

[说明光学系统结构：层状结构、入射方式、探测方式]

物理量定义：
  表X  干涉模型符号说明
  | 符号 | 含义 | 单位 | 类型 |
  |------|------|------|------|
  | d    | 薄膜厚度 | nm | 待求量（目标参数） |
  | n    | 薄膜折射率 | 无量纲 | 已知参数 |
  | n1   | 入射介质折射率 | 无量纲 | 已知参数 |
  | n2   | 基底折射率 | 无量纲 | 已知参数 |
  | lambda | 入射光波长 | nm | 已知参数 |
  | theta_i | 入射角 | ° | 已知参数 |
  | theta_t | 折射角 | ° | 中间变量 |
  | Delta | 光程差(OPD) | nm | 中间变量 |
  | m    | 干涉级次 | 无量纲 | 待定 |

模型机理：
  基于光的干涉理论，研究多层薄膜结构中的光波干涉行为。
  当光在薄膜上下表面反射后相遇时，由于光程差的存在，
  产生干涉现象，其干涉强度反映了薄膜的厚度信息。

  光程差（OPD, Optical Path Difference）：
    Delta = 2 * n * d * cos(theta_t)                        (1)
    其中：2nd 为往返光程，cos(theta_t) 为倾斜因子

  Snell折射定律：
    n1 * sin(theta_i) = n * sin(theta_t)                    (2)

  干涉极大条件（明纹）：
    2 * n * d * cos(theta_t) = m * lambda,  m = 0,1,2,...   (3)

  干涉极小条件（暗纹）：
    2 * n * d * cos(theta_t) = (m + 1/2) * lambda           (4)

  [若为正入射 theta_i = 0°, cos(theta_t) = 1]：
    干涉极大条件简化为：2 * n * d = m * lambda              (3')

  引入波数 nu = 1/lambda，相邻干涉极大间满足：
    2 * n_e * d * Delta_nu = 1                              (5)
    由此可得膜厚公式：
    d = 1 / (2 * n_e * Delta_nu)                            (6)

  [若为Fabry-Perot多光束干涉，补充]：
    透射率 T = T_max / [1 + F * sin^2(delta/2)]             (7)
    其中 F = 4R/(1-R)^2 为精细度系数，R为反射率
    delta = (4 * pi * n * d * cos(theta_t)) / lambda        (8)

参数说明：
  n: 薄膜折射率，取 n = [值]，来源：[文献值/手册/已知材料]
  n_e: 有效折射率，n_e = n * [1 + (lambda/n)*(dn/dlambda)]
       取 n_e = [值]，来源：[文献/手册]
  n1: 入射介质折射率，空气 n1 = 1.0
  theta_i: 入射角，取 theta_i = 0°（垂直入射，简化模型）

模型假设：
  (1) 膜层均匀性假设：薄膜厚度在测量区域内均匀一致
  (2) 理想界面假设：各界面为理想光滑平面，忽略粗糙度引起的散射
  (3) 弱吸收假设：外延层在测量波段内吸收系数足够小，可忽略消光
  (4) 平行光入射假设：入射光为准直平行光，忽略光束发散角
```

**模板B：热传导/扩散模型**

```
X.X 模型建立

[说明物理系统和传热/扩散过程]

物理量定义：
  表X  热传导模型符号说明
  | 符号 | 含义 | 单位 | 类型 |
  |------|------|------|------|
  | T(x,t) | 位置x处t时刻的温度 | K | 因变量 |
  | k | 热导率 | W/(m*K) | 已知参数 |
  | rho | 密度 | kg/m^3 | 已知参数 |
  | c_p | 比热容 | J/(kg*K) | 已知参数 |
  | alpha | 热扩散率 | m^2/s | 导出参数 |
  | Q(x,t) | 热源项 | W/m^3 | 已知/假设 |

控制方程：
  基于Fourier热传导定律和能量守恒原理，建立热传导方程：

  三维热传导方程（一般形式）：
    rho * c_p * dT/dt = div(k * grad T) + Q(x,t)            (1)
    含义：左侧为内能变化率，右侧第一项为热传导项，第二项为热源

  [一维简化，常物性]：
    dT/dt = alpha * d^2T/dx^2                               (2)
    alpha = k / (rho * c_p)                                 (3)

参数说明：
  k: 热导率，取 k = [值] W/(m*K)，来源：[材料手册/文献]
  rho: 密度，取 rho = [值] kg/m^3，来源：[材料手册/文献]
  c_p: 比热容，取 c_p = [值] J/(kg*K)，来源：[材料手册/文献]

初始条件：
  T(x,0) = T_0,  0 <= x <= L                                (4)

边界条件（三选一或组合）：
  [Dirichlet边界]：T(0,t) = T_a, T(L,t) = T_b               (5)
  [Neumann边界]：-k*dT/dx|x=0 = q_0                         (5')
  [Robin边界]：-k*dT/dx|x=0 = h * (T(0,t) - T_inf)          (5'')

模型假设：
  (1) 均匀介质假设：材料热物性参数为常数
  (2) 一维传热假设：横向梯度远小于纵向梯度
  (3) 无相变假设：温度范围内不发生相变
```

### 2.8 物理模型推导路径要求

当模型基于已知物理定律时，**必须**写出完整的推导链条：

```
示例——从物理定律到模型公式的推导路径：

光学干涉模型：
  Maxwell方程组 -> 平面波解 -> 边界条件匹配 ->
  Fresnel公式（振幅反射/透射系数） ->
  光程差(OPD)计算 -> 相位差计算 ->
  干涉叠加（强度叠加公式） -> 干涉强度公式 ->
  极值条件 -> 测量公式 (d = 1/(2*n_e*Delta_nu))

热传导模型：
  Fourier定律 -> 能量守恒 ->
  热传导方程(PDE) -> 分离变量法/Green函数法 ->
  解析解/数值解 -> 温度分布

波动模型：
  Newton第二定律/连续性方程 ->
  波动方程(PDE) ->
  给定边界条件和初始条件 ->
  特征函数展开/分离变量法 ->
  驻波解/行波解 -> 共振频率/传播特性
```

**推导过程中必须做到**：
- 每一步写明"由[物理定律]可得"或"代入[公式]得到"
- 关键近似步骤需特别说明：如小角度近似 sin(theta)≈theta
- 量纲检查：最终公式的量纲必须正确

### 2.9 物理模型常见错误

| 错误 | 示例 | 纠正 |
|------|------|------|
| 只写最终公式不写推导 | 直接给出 d=1/(2n_e*Delta_nu) | 写出：OPD -> 干涉条件 -> 波数转换 -> 膜厚公式 |
| 物理量无定义 | 公式中出现 n, d, theta | 每个符号：名称/单位/含义 |
| 物理定律名称模糊 | "根据光学原理" | 写明："根据Snell折射定律" |
| 缺少近似条件说明 | 直接用 2nd = m*lambda | 说明：仅在正入射(cos(theta_t)=1)时成立 |
| 量纲不检查 | 给出 d = lambda/(2n) | 检查：d[nm]=lambda[nm]/(2*无量纲) OK |
| 参数无来源 | "取 n = 3.5" | 补充：Si在红外波段的折射率，引自文献[XX] |
| 忽略色散效应 | n视为常数 | 如适用，使用Sellmeier公式或限定波段范围 |


## 六、常见错误与纠正

| 错误 | 示例 | 纠正 |
|------|------|------|
| 变量无定义 | "Y = β₀ + β₁X + ε" | 补充：Y为...，X为...，β₁表示... |
| 公式无推导 | 直接列出复杂公式 | 说明推导步骤或引用文献 |
| 无模型类型声明 | "建立数学模型如下" | 改为"建立线性规划模型" |
| 无假设说明 | 使用OLS但未提假设 | 补充线性性/独立性/正态性假设 |
| 参数无来源 | "取α = 0.3" | 说明：α通过[方法]估计/α取[文献]推荐值 |
| 优化模型缺约束 | 只有目标函数 | 补充所有约束条件 |
| 优化模型缺变量范围 | x_i 无定义域 | 补充 x_i ≥ 0 或 x_i ∈ {0,1} |
| 公式与文字脱节 | 公式后直接跳到结果 | 每公式后必须有文字解释 |

## 八、模型假设分类规范（v4.0 新增）

> **来源**: KyrieZhang329/MathModeling-skills model-assumptions-builder

### 8.1 假设分类维度

每个模型假设必须按两个维度分类：

**维度1：必要性**
| 类型 | 定义 | 标志 |
|------|------|------|
| **必要假设（Hard）** | 违反则模型失效或无意义 | "如果不成立，模型将崩溃" |
| **简化假设（Soft）** | 违反则精度下降但模型仍可用 | "为简化计算，近似认为..." |

**维度2：作用范围**
| 类型 | 定义 | 标志 |
|------|------|------|
| **全局假设** | 适用于所有子问题 | "假设所有数据准确可靠" |
| **子问题假设** | 仅适用于特定子问题 | "Q1假设正入射条件" |

### 8.2 假设编写标准模板

```
X.X 模型假设

[全局必要假设]
(1) [假设名称]：[假设内容]。
    必要性：必要假设（Hard）— 如果不成立，[说明后果]。
    理由：[为什么该假设是合理的]。

[全局简化假设]
(2) [假设名称]：[假设内容]。
    必要性：简化假设（Soft）— 放宽此假设会使[说明影响]。
    理由：[为什么可以做此简化]。

[Q1子问题假设]
(3) [Q1专用假设名称]：[假设内容]。
    适用范围：仅Q1。
    必要性：[Hard/Soft]。
    理由：[合理性说明]。
```

### 8.3 反例：不好的假设 vs 好的假设

```
BAD:
  假设1：所有数据都是准确可靠的。
  → 问题：太笼统，没有说明必要性，没有解释为什么可以这样假设

GOOD:
  假设1：数据准确性假设（全局/Soft）
  题目未提供独立的测量误差信息，故将记录值视为最佳可用观测。
  异常值的影响在数据审计和敏感性分析中进行了验证（见第X节）。
  → 明确了必要性和适用范围，给出了替代方案（敏感性分析验证）
```

```
BAD:
  假设2：忽略次要因素。
  → 问题：没说忽略什么因素，为什么是次要的

GOOD:
  假设2：弱吸收假设（Q1/Hard）
  外延层在工作波段（1000-2500 nm）内的消光系数 k < 0.01，
  对干涉条纹对比度的影响可忽略（引自文献[3]）。
  → 具体说明了什么被忽略、数值依据、文献来源
```

### 8.4 自检清单（新增）

- [ ] 每个假设标注了必要性：Hard（必要）或 Soft（简化）
- [ ] 每个假设标注了作用范围：Global 或 Qx
- [ ] 每个 Hard 假设说明了违反的后果
- [ ] 每个 Soft 假设说明了放宽后的影响
- [ ] 避免笼统假设（如"数据准确""忽略次要因素"）
- [ ] 物理参数假设有文献/手册依据
- [ ] 物理模型有：物理量定义 + 控制方程 + 初始/边界条件 + 推导路径
- [ ] 每个物理量已标注单位和量纲
- [ ] 物理定律名称准确（如Snell定律、Fresnel公式、Fourier定律）
- [ ] 近似条件已明确说明（如小角度近似、弱吸收近似）
- [ ] 参数取值有物理依据（文献/手册/计算）

## 九、模型建立扩充清单（v5.1 — 模型太薄时必查）

> **来源**: math-modeling-competition-workflow — 当模型建立章节内容不足 2,000 字时，
> 从以下 6 个维度逐项检查并扩充。**不要灌水，每项扩充必须有数学实质。**

### 9.1 六大扩充维度

| 维度 | 具体操作 | 示例 |
|------|----------|------|
| **1. 数据到模型的映射** | 说明原始数据如何转化为模型输入，画出数据流→模型流的复合流程图 | "原始评分矩阵 S_{ij} 经 Z-score 标准化后作为模型输入 X_{ij}" |
| **2. 损失函数与优化目标** | 明确写出损失函数/目标函数的数学形式，解释为什么选这个形式（而非其他） | "选择 MAE 而非 MSE 的原因：MSE 对离群值过于敏感，与 MAD 检测逻辑不一致" |
| **3. 正则化与约束** | 说明正则化项（L1/L2/弹性网）或约束条件（硬/软）的含义和取值依据 | "贪心算法中回避本校是硬约束，负载均衡是软约束（目标函数项）" |
| **4. 稳健统计方法** | 如果数据含噪声，说明采用的稳健方法：Hampel滤波器、MAD、Huber损失、分位数回归、残差修正等 | "使用 MAD(1.4826) 替代标准差进行离群检测，因为标准差在小样本下被污染数据夸大" |
| **5. 特征工程定义** | 如果涉及特征构造，列出：滞后项、滚动窗口、事件指示器、阶段标签等的精确定义 | "滚动窗口宽度 w=5，覆盖前后各2份答卷，用于计算局部评分均值和波动" |
| **6. 算法伪代码表** | 对核心算法给出：输入、步骤、输出的结构化伪代码 | "算法A.3：三步分数调整。输入：评分矩阵S、评委集J、答卷集P。输出：最终得分Final_i" |

### 9.2 扩充优先级

按以下顺序逐级扩充，每级通过则进入下一级：

```
第1级: 变量定义是否完整？ → 补充缺失的符号、含义、单位、类型
第2级: 公式是否有推导？   → 补充从原理到公式的推导链（至少2步）
第3级: 损失函数是否解释？ → 说明为什么选这个函数（对比至少1个替代方案）
第4级: 约束是否分类？     → 区分硬约束/软约束，标注每个约束的现实含义
第5级: 是否有伪代码？     → 为核心算法添加输入/步骤/输出的结构化伪代码
第6级: 参数是否有依据？   → 每个参数标注来源（数据估计/文献/经验/灵敏度分析）
```

### 9.3 模型建立 vs 模型求解：严格分离

**模型建立章节（禁止放入的内容）：**
- ❌ 数值结果（R²、MAE、AUC等）
- ❌ 验证指标
- ❌ 代码实现细节（除非是伪代码级别的算法描述）
- ❌ "如表X所示"的表格引用（结果表属于求解章节）

**模型建立章节（必须有的内容）：**
- ✅ 模型类型声明
- ✅ 所有变量的符号/含义/单位/类型
- ✅ 完整的数学表达式（目标函数+约束/控制方程+初边值条件）
- ✅ 推导过程或文献依据
- ✅ 参数含义解释
- ✅ 模型假设及合理性论证

**模型求解章节（对应每个子问题）：**
- ✅ 实现来源和可复现路径
- ✅ 求解步骤（按顺序描述）
- ✅ 关键图表（紧接着其支撑的结果放置）
- ✅ 每张图表的三段论分析（描述→分析→结论）
- ✅ 子问题的直接答案

### 9.4 图表三段论规则（每张图/表强制执行）

```
描述 (Description):  图表展示了什么内容
分析 (Analysis):     观察到了什么模式/对比/趋势
结论 (Conclusion):   由此得出什么决策或答案

反例：
  "图3展示了不同L值下的MAE变化。" — 只有描述，没有分析和结论

正例：
  "图3展示了L=2至L=7对应的MAE变化趋势（描述）。

   L从3到4时MAE降低12.8%，而L从6到7时仅降低7.4%（分析）。
   综合考虑成本和精度，选择L=4作为最优评阅人数（结论）。"
```

## 十、模型推导深度标准 (v5.2)

> **来源**: math-modeling-competition-workflow。模型建立必须讲清楚模型是什么、变量是什么、约束和目标是什么、推导如何展开。

### 10.1 推导深度四级标准

| 级别 | 名称 | 要求 | 判据 |
|------|------|------|------|
| **L1** | 公式堆砌 | 只有公式，无解释 | FAIL |
| **L2** | 符号定义 | 公式+符号表+一句话解释 | MINIMAL |
| **L3** | 推导展开 | 公式+推导步骤+每步文字解释 | PASS |
| **L4** | 深度推导 | L3+替代方案对比+选择理由+参数来源 | EXCELLENT |

**目标**: 每个模型至少达到 L3，核心模型达到 L4。

### 10.2 推导展开六步模板

每个模型必须按以下结构展开：

1. **问题形式化**: 用数学语言重述：已知什么？要求什么？约束是什么？
2. **建模思路**: 为什么选择这类模型？考虑过替代方案吗？核心思想一句话概括
3. **变量与参数定义**: 符号表（含义+单位+类型+范围），区分决策变量/状态变量/参数
4. **目标函数/控制方程推导**: 从基本原理出发，至少2步推导，每步配文字解释
5. **约束条件/边界条件**: 每个约束写数学表达式+现实含义+分类（硬/软）+来源
6. **模型假设论证**: 每条假设：条件+合理性论证+违反后果+放松方案

### 10.3 推导展开正例（L3水平）

`
Step 0 - 问题形式化：
  已知：n个候选地点，地点i的建设成本为c_i、覆盖范围为r_i。
  要求：选择一组地点，使总成本最小且所有需求点被覆盖。

Step 1 - 建模思路：
  集合覆盖选址问题。选整数规划的原因：
  (1) 决策变量天然是0-1类型（选/不选）
  (2) 覆盖约束可表示为线性不等式
  (3) 总成本是决策变量的线性函数

Step 2 - 变量定义：
  决策变量：x_i in {0,1}，i=1,...,n
  参数：c_i（万元，来自附件1），a_ij（由距离和覆盖半径r_i计算）

Step 3 - 目标函数推导：
  总成本 = 所有被选地点成本之和
  地点i的成本贡献 = c_i * x_i
  Z = sum_{i} c_i * x_i                    (1)
  min Z

Step 4 - 约束条件推导：
  覆盖约束：sum_i a_ij * x_i >= 1, forall j    (2)
  预算约束：sum_i c_i * x_i <= B                (3)
  变量类型：x_i in {0,1}                      (4)

Step 5 - 模型假设：
  (H1) 覆盖是二值的 — 合理性：简化使模型可解。放松：可引入覆盖度函数f(d_ij)
  (H2) 成本独立 — 合理性：题目未给关联信息

Step 6 - 完整模型汇总：
  min  Z = sum_i c_i * x_i
  s.t. sum_i a_ij * x_i >= 1,  forall j
       sum_i c_i * x_i <= B
       x_i in {0,1},          forall i
`

### 10.4 各模型类型推导锚点

**优化模型**: 必须展示目标函数构建过程、逐一解释约束、说明线性/非线性/整数原因
**统计/回归模型**: 必须通过散点图/相关分析说明函数形式选择、写出最小二乘推导、说明误差分布
**微分方程模型**: 必须从物理机理一步步建立方程、说明每项物理含义和量纲、说明初边值条件来源
**评价模型**: 必须说明指标体系构建理由、解释权重确定方法、说明综合评分合成方式
**概率/随机模型**: 必须说明分布选择依据、展示参数估计过程、说明随机性来源

### 10.5 推导充分性自检清单

写完每个模型后，逐项自检：

- [ ] 是否从已知什么/要求什么开始写？
- [ ] 每个变量是否有符号+含义+单位+类型？
- [ ] 每个公式是否至少配了2句文字解释？
- [ ] 目标函数是否有至少1步推导（不是直接写出）？
- [ ] 每个约束条件是否有现实含义的解释？
- [ ] 关键参数是否有来源标注？
- [ ] 模型假设是否有如放松则…的说明？
- [ ] 是否说明了为什么选这个模型类型？
- [ ] 完整模型是否汇总到了一起？
- [ ] 推导步骤间是否有逻辑连接词？

# 五一数学建模竞赛 (51MCM) 论文格式规范

基于《2026年五一数学建模竞赛论文格式规范》官方文档提取。
所有 DOCX 生成必须严格遵循此规范。

## 论文结构（按页）

| 页码 | 内容 | 说明 |
|------|------|------|
| 第1页 | 承诺书 | 使用模板格式，不超过1页 |
| 第2页 | 题目 + 摘要 + 关键词 | 摘要不超过1页 |
| 第3页起 | 正文 | 无目录，尽量≤30页 |
| 正文后 | 参考文献 | 按引用次序列出 |
| 最后 | 附录 | 源程序代码 + 模型结果获取说明 |

禁止项：
- 无页眉
- 无任何显示答题人身份的标志
- 无目录

## 字体规范

| 元素 | 字体 | 字号 | 对齐 | 加粗 |
|------|------|------|------|------|
| 论文题目 | 黑体 | 三号 (16pt) | 居中 | 是 |
| 一级标题 | 黑体 | 四号 (14pt) | 居中 | 是 |
| 二级标题 | 黑体 | 小四 (12pt) | 左对齐 | 是 |
| 三级标题 | 黑体 | 小四 (12pt) | 左对齐 | 是 |
| 正文汉字 | 宋体 | 小四 (12pt) | 两端对齐 | 否 |
| 正文拉丁字母 | Times New Roman | 小四 (12pt) | — | 否 |
| 摘要标题 | 黑体 | 四号 (14pt) | 左对齐 | 是 |
| 摘要正文 | 宋体 | 小四 (12pt) | 两端对齐 | 否 |
| 关键词 | 黑体(标签) 宋体(内容) | 小四 (12pt) | 左对齐 | 标签加粗 |
| 表格标题 | 黑体 | 五号 (10.5pt) | 居中 | 是 |
| 表格内容 | 宋体 | 五号 (10.5pt) | 居中 | 否 |
| 图片标题 | 黑体 | 五号 (10.5pt) | 居中 | 是 |
| 参考文献正文 | 宋体 | 小四 (12pt) | 左对齐 | 否 |
| 附录标题 | 黑体 | 四号 (14pt) | 居中 | 是 |

## 段落规范

- 行距：单倍行距（1.0）
- 首行缩进：2字符（约24pt for 小四号）
- 段前/段后：0（正文），适当间距（标题）

## 表格规范（三线表）

三线表定义：
- 顶线：1.5pt 粗实线
- 表头下线：0.5pt 细实线  
- 底线：1.5pt 粗实线
- 无竖线，数据行之间无横线
- 表头：黑体 五号 加粗 居中
- 表体：宋体 五号 居中

## 公式规范

- 使用 OMML (Word原生Math) 或 AxMath 渲染，不可用纯文本
- 公式居中显示
- 公式编号右对齐：(1), (2), ...
- 变量符号：Times New Roman 斜体
- 常量/单位：Times New Roman 正体

## 图片规范

- 分辨率 ≥ 300 DPI
- 图题：黑体 五号，居中，置于图下方
- 格式：图1, 图2 ...（中文编号）
- 坐标轴标签清晰可读

## 参考文献规范

引用方式：正文中用方括号 [1][3] 标示编号。
参考文献按引用次序列出。

书籍格式：
  [编号] 作者, 书名, 出版地: 出版社, 出版年.

期刊格式：
  [编号] 作者, 论文名, 杂志名, 卷期号: 出版年, 起止页码.

网上资源格式：
  [编号] 作者, 资源标题, 网址, 访问时间（年月日）.

## AI 工具使用规定

- 须在附录中声明使用的 AI 工具名称、版本、用途
- 格式：工具名称与版本 / 使用目的与环节 / 使用方式说明

## 提交要求

- 最终提交：PDF 文档（非纯图片、未加密）
- 中间过程：DOCX 便于编辑

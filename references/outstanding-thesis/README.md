## 使用指南

### 如何阅读论文

使用 **pdf skill** 读取PDF格式的优秀论文：

```
1. 确定需要参考的论文类型
2. 使用 pdf skill 读取对应的PDF文件
3. 学习论文的写作技巧和方法
```

### 学习重点

| 学习内容 | 说明 |
|---------|------|
| 论文结构 | 章节安排、逻辑顺序 |
| 摘要写作 | 如何突出亮点 |
| 模型表述 | 公式规范、符号定义 |
| 图表设计 | 类型选择、标注规范 |
| 结果分析 | 深入分析、模型检验 |

### 注意事项

- ⚠️ 仅供学习参考，请勿直接复制
- ⚠️ 理解方法后用自己的方式表达
- ⚠️ 论文格式可能随年份有所变化
